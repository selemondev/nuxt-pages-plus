# nuxt-pages-plus

## Contribution

<details>
  <summary>Local development</summary>
  
  ```bash
  # Install dependencies
  pnpm install
  
  # Generate type stubs
  pnpm dev:prepare
  
  # Develop with the examples
  pnpm dev examples/...

  # Develop with the playground
  pnpm play
  
  # Build the playground
  pnpm play:build

  # Run ESLint
  pnpm lint
  
  # Run Vitest
  pnpm test
  pnpm test:watch
  
  # Release new version
  pnpm release
  ```

</details>
